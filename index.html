Projets

GestionCommerciale

  • Objectif : Développer une application de gestion commerciale en Java en consommant des API Symfony pour la gestion des produits et des commandes.
  • Technologies utilisées : Java, JavaFX, Symfony, API REST
  • Contribution : J'ai conçu l'interface utilisateur en JavaFX, intégré les API, et implémenté les fonctionnalités CRUD pour la gestion des commandes.
  • Défis : J'ai dû sécuriser les échanges de données entre l'application et les API, et j'ai implémenté des tests unitaires pour valider les interactions avec l'API.
  • Résultat : L'application fonctionne et est utilisée pour la gestion commerciale au sein du projet "VillageGreen".
